PHP 代码执行的危险函数

eval()
传入命令需要以分号结尾

assert()
传入命令不需要分号结尾

preg_replace()

create_function()
需要加分号

array_map()
array_map(func , arg)
func传入函数， arg传入参数数组

call_user_func()
传入参数作为assert函数参数

call_user_func_array()
传入参数作为数组第一个值给assert函数

array_filter()
用回调函数过滤数组中元素:array_filter(数组， 函数)

uasort()
php环境>=<5.6可以使用

PHP 命令执行函数

system();
可以直接显示结果

passthru();
可以直接显示结果

shell_exec();
函数可执行，但是要加echo 回显

exec
(可以执行命令，以数组形式的保存结果，命令执行的状态码)
但需要加echo才能显示结果

popen
//popen函数：打开一个指向进程的管道，该进程由派生指定的 command 命令执行而产生。
//返回一个和 fopen() 所返回的相同的文件指针，只不过它是单向的（只能用于读或写）
//此指针可以用于 fgets()，fgetss() 和 fwrite()。并且必须用 pclose() 来关闭。
//若出错，则返回 false。

proc_open
//proc_open — 执行一个命令，并且打开用来输入/输出的文件指针。

``反斜杠
本质上是 shell_exec函数,如果shell_exec禁用，该函数同样不能使用
